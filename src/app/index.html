What is OpenShift.io (OSIO)?

It is an online development environment for planning, creating and deploying hybrid cloud services. OpenShift.io provides an effortless approach to DevOps, a one-click Linux container environment for developers and better decision making through machine learning. Through a fully integrated set of development tools with OpenShift Online, OpenShift.io reduces the time and risk of deploying modern applications. OpenShift.io is a year-old open source project, and incorporates many other open source projects including fabric8, Eclipse Che, OpenJDK, PCP, WildFly Swarm., Vert.x, Spring Boot and of course, OpenShift.

We want to support customers and partners by minimizing the time and effort it takes to build and maintain an end-to-end development tool chain.

OpenShift.io It is a hosted service with planning tools like like prioritizable backlogs and kanban boards for managing your work. It has coding editing and debugging tools built on Eclipse Che that provide developers with a full featured IDE. We provide dashboard and reporting tools that give teams and developers information about how their projects are progressing and how their code is performing. All of these tools are integrated by default and share a common datastore.

These functionalities are just a few of the capabilities provided to drive developer efficiency.

We want to reduce the time and effort it takes to create containerized dev, test and staging environments that plague developers today when using bare metal, virtual machines or tools like Vagrant and Docker.

OpenShift.io integrates the Jenkins Pipeline plugins to give developers a complete library from which to assemble their pipeline, from the simplest ‘just build it’ to support for promotions between environments and human approval steps. As pipelines are written using a Groovy DSL you have huge flexibility in how you build the pieces into a couple of pipelines.

Automatically create containerized development environments. We do this by incorporating the workspace management capabilities of Eclipse Che. This creates environments, using your favorite technology stacks, that are hosted in a container on OpenShift online.

We want to enable development teams be more be more confident in the technology decisions they make.
